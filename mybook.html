        course database table


        course-Id (primary key)....................................: Id for the course

        5. This table contain all work or assignment created by admin and admin give permission all students to get that work easily in the short time. And this contain name of assignment, title, id of course and time of submission of that assignment. .

         assignment table has the following attributes

        Assignment database table


        assignment-Id (primary key)...................................: Id for the assignment
        assignment_mame (varchar)............................: name of assignment
        course-id (foreign key)..............................: id of course
        assignment-title (varchar).....................................: title of assignment
        deadline (varchar)......................................: Time of submission
        maximum-score (int)...........................: marks of assignment

        6. This table contain all exam prepared by admin for all student .name of that exam, id of that, time of starting exam and time of finishing it. Admin will provide username and password for candidates so that they can login and start exam.

         exam table has the following attributes

        exam database table


        exam-Id (primary key)......................................: Id for the exam
        exam-name (foreign key)..................................: name of exam
        course-id (foreign key)............................: id of course
        start-time(varchar).................................:time of starting exam
        end-time(varchar).....................................:time of finishing exam
        exam-title(varchar).....................................:title of exam

      6. enrollment Table

        7. This table contain id of student enrolled, id of that user and course id of that lesson. This keeps tracks of which classes each students is enrolled in and links the class id and students id. .

         enrollment table has the following attributes

        enrollment database table


        enrollment-id (primary key)......................................: Id for the enrolled
        user-id (foreign key)............................................: id of the user
        course-id (foreign key)...........................: id of the course

      7. submission Table

        8. This table contain id of submitted, id of submitter, id of that thing submitted, time of submission and marks of that things. For this students are allowed to submit what he/she done and allowed to get marks of that.

        submission database table


        submission-id (primary key)......................................: Id for the submission
        user-id (foreign key)............................................: id of the user
        assignment-id (foreign key)...........................: id of the assignment
        submission-time(varchar).............................................:time of submission
        score(int)............................................................:marks of course submitted

      8. grade Table

        9. This table contain id of grade, user id, assignment id and exam id. For this student allowed to get grade of what he/she submit.

        grade database table


        grade-id (primary key)......................................: Id for the grade
        user-id (foreign key)............................................: id of the user
        assignment-id (foreign key)...........................: id of the assignment
        exam-time(foreign key).............................................:time of exam
